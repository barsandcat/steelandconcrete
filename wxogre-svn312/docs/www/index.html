What is it?

wxOgre is a little library allowing you to easily create applications mixing Ogre and wxWidgets. It is written in C++, runs on Windows and GNU/Linux, is free, open source and open to contributions.

Get it!

You can download wxOgre 0.2 (zip, gz) from SourceForge's repositories or checkout the development trunk: 

svn co https://owlet.svn.sourceforge.net/svnroot/owlet/wxogre/trunk wxogre
