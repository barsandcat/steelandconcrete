1. Introduction

This tutorial is the main source of documentation about wxOgre itself. It is not so long and very example oriented. Even if advanced knowledge of Ogre and wxWidgets aren't really necessary they will you help because I wont explain any specific stuff about those libraries.

Source code

The source code of those tutorials can be found in the example folder in the source archive and is not covered by the LGPL license. You can do what you want with it.

Naming convention

Like for the library, I try to follow wxWidgets naming convention. It is why Class methods start with an upper case letter and private class variables are in lower case prefixed by m_:

Exemple 1.1

m_rsys->LoadPlugin("RenderSystem_GL");

As you can see, my code is four space indented and, because I code in a terminal, lines are 80 chars long.

Improving this tutorial

Like for the library you can contribute to this tutorial sending me comments, correction or new parts. The code of the tutorial is also included in the source archive and can be retrieve from the svn repository: 

svn co https://owlet.svn.sf.net/svnroot/owlet/wxogre/docs/www/tutorial

Ok, and now, ready to write your first wxOgre application? Let's code!
